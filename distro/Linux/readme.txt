===========
= Install =
===========
1. Unzip OnsenlibSynth-Linux.zip
2. Copy OnsenlibSynth.vst3 to /usr/lib/vst3/ (sudo cp -r OnsenlibSynth.vst3 /usr/lib/vst3/)
3. Copy OnsenlibSynth.lv2 to /usr/lib/lv2/ (sudo cp -r OnsenlibSynth.lv2 /usr/lib/lv2/)

=============
= Uninstall =
=============
1. Delete /usr/lib/vst3/OnsenlibSynth.vst3 (sudo rm -rf /usr/lib/vst3/OnsenlibSynth.vst3)
2. Delete /usr/lib/lv2/OnsenlibSynth.lv2 (sudo rm -rf /usr/lib/lv2/OnsenlibSynth.lv2)
